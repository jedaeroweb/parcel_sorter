## 🔭 Overview

https://sorter.jedaeroweb.co.kr

**Package Sorter Puzzle** is a free browser game in which players sort packages flowing on a conveyor belt within a limited time.

Built with **Next.js** and **TypeScript**, the game supports multiple languages and runs directly in modern web browsers without installation.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/jedaeroweb/parcel_sorter.git
cd YOUR_REPOSITORY
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

or:

```bash
yarn dev
```

or:

```bash
pnpm dev
```

or:

```bash
bun dev
```

Open:

```text
http://localhost:3000
```

The application will automatically reload when files are modified.
